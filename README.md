# Neovim Keybinds Documentation

This document provides a simple and organized overview of all the custom keybinds defined in my Neovim configuration.

## Installation

### Linux and macOS

```
git clone https://github.com/ramzt3/nvim-12 "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

### Windows
If you're using `cmd.exe`:
```
git clone https://github.com/ramzt3/nvim-12 "%localappdata%\nvim"
```
If you're using `powershell.exe`:
```
git clone https://github.com/ramzt3/nvim-12 "${env:LOCALAPPDATA}\nvim"
```
